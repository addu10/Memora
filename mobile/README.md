# Memora Mobile App

A premium React Native (Expo) mobile application for Alzheimer's patients to engage in memory therapy sessions, featuring a soothing light-themed UI and AI-driven reminiscence therapy.

## ğŸŒŸ Overview

The Memora Mobile App is designed for elderly patients with early-stage Alzheimer's disease. It prioritizes accessibility, visual comfort, and cognitive engagement through:

- **Premium Light Theme**: High-contrast "White-Lavender" aesthetics with glassmorphism and smooth gradients.
- **Face Recognition (V2)**: Instant family identification using DeepFace and Supabase Edge Functions.
- **AI-Powered Therapy**: Personalized interactive prompts generated by AI to stimulate memory.
- **Slideshow Mode**: Soothing, auto-advancing memory slideshows with ambient transitions.
- **Memory Performance**: Visual tracking of recall progress (Mastered, In Progress, Needs Practice).
- **Proactive Safety**: Global Error Boundaries and Network Connectivity monitoring.

## ğŸ” Authentication Flow

1. **Secure Access**: Patients log in using a 4-digit PIN matched via `login_patient` RPC.
2. **Session Persistence**: Authentication state is managed via Supabase and cached in `AsyncStorage`.
3. **Safety First**: Any failed attempt provides graceful feedback via standardized `StatusModals`.
