# Fallback Questions Reference

Use this document to identify if questions displayed in the therapy session are **AI-generated** or **fallback questions**.

> **If you see any question below, the AI generation failed and fallback was used.**

---

## Set 1 (Photo Index: 1, 5, 9...)

| Type | Question | Hint |
|------|----------|------|
| Contextual | What's the first thing you notice in this photo? | Look at the colors and shapes around you. |
| Relational | Tell me about [Person]. What's a good memory you share? | Think of times with [Person]. |
| Relational (no person) | Who comes to mind when you see this? | Someone who'd enjoy being here. |
| Emotional | If you could relive this moment, what would you do? | What made this time special? |

## Set 2 (Photo Index: 2, 6, 10...)

| Type | Question | Hint |
|------|----------|------|
| Contextual | It looks like [activity]. What was that experience like? | How did [activity] feel? |
| Contextual (no activity) | Describe what's happening here. | What do you see people doing? |

| Relational (no person) | Who organized this gathering? | Think about who made it happen. |
| Emotional | Everyone seems [expression]. What created that feeling? | Any feeling is valid. |
| Emotional (no expression) | What emotions does this photo bring up? | Any feeling is valid. |

## Quick Identifiers

**Common fallback phrases to look for:**
- "What's the first thing you notice..."
- "Tell me about [Person]. What's a good memory..."
- "If you could relive this moment..."
- "It looks like [activity]. What was that experience..."
- "How did you and [Person] first meet?"
- "Everyone seems [expression]. What created that feeling?"
- "This [setting] looks special. What sounds..."
- "What's something about [Person] that makes you smile?"
- "What would you tell someone who wasn't there..."
- "What time of year does this feel like?"
- "If this photo could speak..."

**AI-generated questions will:**
- Be more specific to the photo context
- Reference unique details from the memory description
- Have varied sentence structures
- Not match any pattern above exactly
